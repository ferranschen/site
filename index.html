Research Projects


Web Automation for Data Collection

We introduce a mixed-initiative web automation system that helps users create web scraping programs automatically. It is superior to existing tools, as it provides step-by-step explanations with visuals, allows more control, and automatically finds corner cases. Additionally, it has undo/redo features for greater freedom. A study with 24 people showed that our tool reduced task completion time by half and made users more confident in the generated script compared to a state-of-the-art web automation tool.

Keywords: Programming by Demonstration, Web Automation, Data Science

Understanding the Coding Behavior of Data Scientists at the Action Level

Data scientists play an important role in getting useful information from data in many businesses and research areas. Even though data science is used a lot, we don't know much about how data scientists code. Previous studies got code from GitHub and Kaggle, but this only showed the finished product. Our study looked at all the running history and error logs to get a better understanding. We found out the mistakes data scientists make, how they try to fix them, and the patterns of coding and debugging. Finally, we compared our results to previous studies and showed how better tools can help data scientists.

Keywords: Data Science, Jupyter Notebook, Programming Behavior
